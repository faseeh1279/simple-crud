### Installation 
```bash 
npm install --force
```


### Make Migrations 
```bash 
 npm run typeorm migration:generate src/migrations/CreateProductsTable
```

### Migrate 
```bash 
npm run typeorm migration:run
```

### Run the Development Service 
```bash 
npm run start:dev 
```

### Paste the following on Browser 
```bash 
http://localhost:3000/api
```
